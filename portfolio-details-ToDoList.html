ToDoList (Android Studio)

  • Langages : Java
  • Type de réalisation: Réalisation personnel

Description

Ce projet a été réaliser dans le but de m'améliorer en développement mobile, il a été réaliser hors du milieu scolaire et professionnel.
Cette application est une application de To Do List qui sont enregistré dans un fichier interne au téléphone ce qui a pour bénéfice de ne pas avoir besoin d'accès internet pour visualiser les listes.
Les listes peuvent être partagées, de plus il est possible de créer un rappel dans l'agenda google.
Il est possible d'ajouter des recettes prédéfinies aux listes.
