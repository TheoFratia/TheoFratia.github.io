Site Git

  • Langages : HTML, CSS
  • Type de réalisation : en cours de formation
  • Categorie : Gérer le patrimoine informatique

Description

Ce projet a été réaliser durant les cours de première année de BTS SIO.
Le projet est un site qui explique comment utiliser git, ce site à été créé avec git pour apprendre à le prendre en main et pour gérer la sauvegarde des versions de code pour pouvoir les retouver si lors de l'implémentation de nouvelle fonctionnalité cela fait disfonctionner le projet.


Compétences acquises

  • Utilisation de HTML et CSS
  • Versioning du site via GitLab
