Pokemon Bros

  • Language : C#
  • Type de réalisation : en cours de formation

Description

Ce Jeux est un jeu créé en C# grâce à Unity lors d'un cours de mon Diplôme d'études collégiales.
Poukemon Bros est un platformer 2D avec différents ennemies et différents univers a travers les différents niveaux.

Compétences acquises

  • Utilisation de la physique 2D d'Unity
  • Utilisation de PlayerPref pour l'enregistrement de certaines données
  • Annimation 2D
