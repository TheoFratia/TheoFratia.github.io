Utilisation De GitLab

Description

Ce projet a été réaliser dans l'entreprise Département De La Loire au cour du stage de deuxième année de BTS SIO.
L'utilisation de GitLab permet de pouvoir travailler a plusieur sur le même projet mais aussi a avoir des sauvegarde de version de code qui fonction pour pouvoir les retouver si lors de l'implementation de nouvelle fonctionnalité cela fait buguer le projet.
J'ai pu mettre en place un GitLab pour travailler a avec un autre stagiaire sur les projets Carte Collège, Map Loire et pour un backoffice pour crée de nouveau point pour la carte Loire.
